# Personal Blog

A blog based on `next.js`

## Server

1.1 install MySQL

1.2 create db named personal-blog

1.3 implement .sql personal-blog.sql

## Usage

1. install dependence

`npm install` 

2. run

2.1 run dev

`npm run dev`

2.2 run build

`npm run build`

`npm run start`

## automatic deployment

1. install dependence

`npm install`

`npm i pm2 -g`

2. run

`npm run prod`
